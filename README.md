# user-crud-system

Projeto em Java desenvolvido para praticar lógica de programação e programação orientada a objetos (POO), utilizando cadastro e gerenciamento de usuários via terminal.

## 🚀 Funcionalidades

- Cadastro de usuários com diferentes níveis de informações
- Validação simples de e-mail
- Definição de senha (CPF ou senha personalizada)
- Listagem de usuários cadastrados
- Remoção de usuários com validação de senha

## 🧠 Conceitos aplicados

- Programação Orientada a Objetos (POO)
- Encapsulamento
- Manipulação de listas com ArrayList
- Entrada de dados com Scanner
- Estrutura de decisão e repetição


## 📌 Objetivo

Este projeto foi desenvolvido com foco em prática e evolução de lógica de programação, simulando um pequeno sistema de gerenciamento de usuários em ambiente de terminal.
